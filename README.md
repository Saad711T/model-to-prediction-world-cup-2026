# World Cup 2026 Prediction Model

Advanced AI model used to predict world cup 2026 matches results with `JSON` local datasets

![image](https://upload.wikimedia.org/wikipedia/en/thumb/1/17/2026_FIFA_World_Cup_emblem.svg/250px-2026_FIFA_World_Cup_emblem.svg.png)

### Project Structure

- [feature_columns.json](feature_columns.json) : Local Dataset to features used in ML
- [predictions.json](predictions.json) : JSON file contains a results
- [tactical_lstm.pth](tactical_lstm.pth) : PyTorch version of model
- [world_cup_model.pkl](world_cup_model.pkl) : Pickle file for model
